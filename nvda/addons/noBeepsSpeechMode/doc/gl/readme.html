Este complemento exclúe os pitidos no modo de fala cando se prema NVDA+s. Só podes cambiar entre desactivado ou falar.

Cambios para 2.0

  • A Axuda deste complemento está disponible dende o Administrador de Complementos.

Cambios para 1.5

  • Engadido script categoría (falar).
  • Búscase unha versión vella do complemento que non está soportada pola comunidade, e avísase ó usuario antes de retirala.

Cambios para 1.4

  • A partires desta versión, as mensaxes a cerca do modo de voz tradúcense automáticamente dende o NVDA na lingua correcta, xa non se require dunha traducción específica para o complemento (ca posible excepción obvia do manifest.ini e da documentación do script).
